# Глаз художника [Artist's Eye]

**Палочка, необычная**
**Рекомендованная стоимость:** tbd
**Источник:** «Hexbound»

Эту **палочку** можно принять за красивую кисть для рисования. Когда её щетинки не отягощены краской, они словно слегка движутся сами по себе, как будто кисть живая

Держа **Глаз художника**, вы обладаете художественным навыком создавать картины точно такими, какими вы их себе представляете, независимо от вашего реального уровня мастерства. Вы можете, например, создавать фотореалистичные изображения по чётким воспоминаниям, если потратите необходимое количество времени, которое потребовалось бы на их создание

**Палочка** не предоставляет краску или какие-либо другие материалы, их необходимо получить отдельно, чтобы использовать **палочку** для рисования
