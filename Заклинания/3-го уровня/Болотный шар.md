# Болотный шар [Mireball]
3 уровень, [[Вызов]]
Время накладывания: **действие**
Дистанция: **150 футов**

Источник: «Ryoko's Guide the Yokai Realms»

Сгусток липкой грязи с хлюпающим звуком вылетает из вашего пальца в выбранную вами точку, где с влажным чавканьем взрывается, разбрасывая жижу. Все существа в **сфере радиусом 15 футов** с центром в этой точке должны совершить [[Заклинания#Спасброски|спасбросок Силы]]. При **провале** цель получает **3к10 дробящего урона** и [[ДМовские заметки#Опрокинутый|сбивается с ног]]. При **успехе** она получает **половину урона** и не сбивается с ног

Все поверхности в **сфере** до конца заклинания покрываются непредсказуемо скользкими и липкими пятнами грязи. **В первый раз за ход**, когда существо перемещается по грязной области, оно должно преуспеть в [[Заклинания#Спасброски|спасброске Ловкости]], иначе [[ДМовские заметки#Опрокинутый|падает ничком]]. Существо, перемещающееся по области с **половинной скоростью**, не обязано совершать этот [[Заклинания#Спасброски|спасбросок]]

**_На больших уровнях_:** **Урон** увеличивается на **1к10 за уровень**
